* Chat Messenger Project *

#Description
The Chat Messenger Project is a Java-based client-server communication application developed using Socket Programming. 
It allows real-time text communication between multiple users over a network using TCP protocol.

#Technologies Used
1) Java
2)Socket Programming
3)TCP/IP Protocol
4)Multithreading
5)Java I/O Streams

#How It Works
1)The server starts and listens on a specific port.
2)Clients connect using IP address and port number.
3)Messages are sent and received in real-time.
4)Communication continues until user exits.
